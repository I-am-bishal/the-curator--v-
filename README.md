# 🧠 The Curator

### *Your Personal Digital Knowledge Manager (CLI आधारित)*

---

## 📌 Overview

**The Curator** is a powerful Python CLI application that helps you **store, organize, and analyze your ideas, notes, and knowledge** in one place.

It acts like a **digital second brain** — automatically categorizing your entries and giving useful insights.

---

## 🚀 Features

* 📥 Add notes, ideas, or links
* 🧠 Automatic categorization (Programming, Finance, Education, etc.)
* 🔍 Search entries by keyword
* 🗂️ Filter by category
* 📊 Insights (most used category)
* 🗑️ Delete entries
* 📁 Export data to CSV
* 💾 Data stored locally using JSON

---

## 🛠️ Tech Stack

* **Language:** Python 🐍
* **Storage:** JSON
* **Libraries Used:**

  * `json`
  * `datetime`
  * `os`

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/the-curator.git
```

2. Navigate to the folder:

```
cd the-curator
```

3. Run the program:

```
python the_curator.py
```

## 💡 How It Works

* User inputs a note or idea
* System automatically assigns a category
* Data is stored with timestamp and username
* You can search, filter, and analyze anytime
